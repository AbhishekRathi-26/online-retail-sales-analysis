# 📊 Online Retail Sales Analysis Dashboard

An end-to-end Data Analytics project built using **Excel, MySQL, and Tableau** to analyze retail sales performance, customer behavior, product performance, and geographical revenue distribution.

---

## 📌 Project Overview

This project uses the **Online Retail Dataset** from the UCI Machine Learning Repository, representing transactions of a UK-based online retail business specializing in giftware and home décor products.

The objective of this project was to clean, analyze, and visualize retail transaction data to generate actionable business insights through an interactive Tableau dashboard.

---

## 🛠️ Tools Used

- Microsoft Excel
- MySQL
- Tableau Public

---

## 📂 Project Workflow

1. Data Cleaning in Excel
2. Data Import into MySQL
3. SQL Analysis using Aggregate Functions
4. Dashboard Development in Tableau
5. Business Insight Generation
6. Documentation

---

## 📈 Dashboard KPIs

- Total Revenue
- Total Orders
- Total Customers
- Average Order Value (AOV)

---

## 📊 Dashboard Visualizations

- Monthly Revenue Trend
- Revenue by Quarter
- Revenue by Country
- Top Revenue Generating Products

---

## 💡 Key Business Insights

- United Kingdom generated the highest revenue.
- Q4 was the strongest performing quarter.
- A small number of products contributed significantly to overall revenue.
- Revenue shows a noticeable increase during the holiday season.
